# Mouse Battery Widget

A minimal Windows desktop widget that displays your wireless mouse 
battery level in real time and alerts you when the battery is low.

## Features
- Real-time battery level display
- Low battery alert notification
- Minimal and clean UI

## Requirements
- Windows 10 or later
- .NET 8
- A wireless mouse with HID battery reporting support

## Installation
1. Download the latest release
2. Run MouseBatteryWidget.exe
3. The widget appears on your desktop automatically

## Technologies
- C# / .NET
- WPF (Windows Presentation Foundation)
- HidLibrary

## Status
Work in progress — UI complete, battery reading in development.
